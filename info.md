Prompt à donner à l’IA

Je veux créer un site web officiel pour la Convention de la Jeunesse Foursquare Côte d’Ivoire.

Le site doit être moderne, professionnel, spirituel, dynamique et adapté à une jeunesse chrétienne. L’objectif est de présenter la convention, informer les participants, permettre l’inscription en ligne, générer un badge avec QR code, et fournir un espace d’administration pour gérer les données.

La stack technique choisie est :

Laravel
Inertia.js
React
TypeScript
Tailwind CSS
Motion for React
MySQL
Filament pour l’administration
Ziggy pour les routes

Le site doit être conçu comme un monolithe moderne Laravel + React, avec Laravel pour le backend et React pour l’interface via Inertia.js.

Pages publiques attendues

Le site doit contenir les pages suivantes :

Accueil
À propos
Programme
Intervenants
Galerie
Inscription
Contact

La page d’accueil doit avoir un design fort avec :

un hero moderne
le nom de la convention
un slogan spirituel
la date
le lieu
un bouton “S’inscrire maintenant”
un bouton “Voir le programme”
des images de jeunes en adoration
des couleurs bleu, orange, vert et blanc

Le style doit être inspirant, propre et premium. Il ne faut pas un design trop enfantin ou trop chargé.

Animation d’ouverture

Je veux une animation simple au chargement du site.

L’idée :

une grande lettre C apparaît au centre de l’écran
ensuite le mot “Convention” se complète
puis “de la Jeunesse” apparaît
ensuite “Foursquare Côte d’Ivoire”
enfin un logo simple apparaît
puis la page d’accueil s’affiche

Le fond doit être bleu nuit ou bleu profond, avec une lumière douce orange/dorée.

L’animation doit durer environ 2 à 2,5 secondes maximum.

Elle doit être :

simple
fluide
élégante
spirituelle
moderne
professionnelle

Elle ne doit pas être trop flashy, trop longue ou trop bruyante.

Fonctionnalités principales

Le site doit permettre :

l’inscription des participants
la gestion des participants
la génération de badge PDF
la génération d’un QR code unique par participant
le scan du QR code à l’entrée
la validation de présence
l’export des participants en Excel
l’export en PDF
la gestion du programme
la gestion des intervenants
la gestion de la galerie
la gestion des messages de contact

Le QR code ne doit pas contenir les informations personnelles du participant. Il doit seulement contenir un identifiant unique sécurisé.

Espace admin

L’administration doit permettre de gérer :

participants
programme
intervenants
galerie
messages de contact
statistiques
exports PDF
exports Excel
scan QR code

Le dashboard admin doit afficher :

nombre total d’inscrits
nombre de participants validés
nombre d’églises représentées
nombre d’intervenants
dernières inscriptions
statistiques simples

L’administration doit être protégée par authentification.

Design attendu

Le design doit être :

moderne
responsive
premium
spirituel
jeune
clair
professionnel

Palette recommandée :

bleu profond
orange
vert
blanc
touches dorées légères

Les animations doivent être faites avec Motion for React.

Animations souhaitées :

apparition douce des sections au scroll
cartes programme qui montent légèrement
boutons avec micro-interactions
hero animé doucement
transition fluide entre les éléments
animation QR code avec effet scan

Il faut éviter les animations excessives.

Hébergement

Le projet doit pouvoir être hébergé sur un hébergement cPanel comme LWS, avec :

PHP
MySQL
SSL
emails professionnels
build Vite généré en production

Le frontend React doit être compilé avec Vite et servi par Laravel.

Résultat attendu

Je veux obtenir un site complet, propre et sérieux pour présenter la convention et gérer les inscriptions.

Le rendu doit donner l’impression d’une vraie plateforme officielle, pas d’un simple site vitrine basique.

Le projet doit être organisé, maintenable, sécurisé et professionnel.
